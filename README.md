# Assignment 1: Multi-Layer Perceptron for Image Classification

This project implements a simple multi-layer perceptron (MLP) using NumPy for MNIST and Fashion-MNIST classification.

## Features
- Configurable network: layers, activation, optimizer, loss
- Training and inference scripts
- Evaluation metrics: Accuracy, Precision, Recall, F1-score
- Weights & Biases (wandb) integration for experiment tracking

## Usage
- Train: `python src/train.py [args]`
- Inference: `python src/inference.py --model_path models/bestmodel.npy [args]`
